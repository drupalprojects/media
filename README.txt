INSTALLATION
-------------------------------------------------------
Place module in your modules directory
Go to admin > build > modules and enable media module

CONFIGURATION
-------------------------------------------------------
Go to admin > content > media 
Configure settings per content type. Each field in a node can
have options for what resources and formatters are available 
to it.
